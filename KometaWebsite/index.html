UI Designer

Requirements for this position:

  • 2+ years of relevant experience;
  • A portfolio showcasing UI work for game projects with casual graphics;
  • Understanding of UI/UX design principles;
  • Proficiency in Adobe Photoshop;
  • Proficiency in Figma;
  • Strong artistic skills in creating icons, banners, and UI elements in a casual style.

Responsibilities:

  • Creating UI for mobile game projects with casual-style graphics based on provided specifications;
  • Designing layouts and preparing state previews;
  • Creating raster game icons;
  • Maintaining a library of UI elements.

Advantages of working with us:

  • Option to work under a Cyprus-based contract with salary tied to EUR, or as an independent contractor (with vacation/holidays aligned to standard labor practices);
  • An experienced team with a strong track record of successful releases;
  • Remote work with a flexible schedule.
